# sfc-gh-jparrado / HOL

Repositorio público con Workshops de Snowflake.

## Workshops disponibles

### 🚀 AI Summit — `AI_SUMMIT/`

Workshop de 20 minutos que muestra capacidades AI de Snowflake: multimodal (imágenes, PDFs, audio), Cortex Analyst, Cortex Search, Snowflake Intelligence Agent y Cortex Code.

**Para instalar:** ejecuta el script en [`INSTALL.md`](./INSTALL.md) con rol `ACCOUNTADMIN`. Tarda ~75 segundos.

Archivo entry point: [`AI_SUMMIT/bootstrap.sql`](./AI_SUMMIT/bootstrap.sql)

> 🤖 **Para agentes LLM (Cortex Code, etc.):** ver [`INSTALL.md`](./INSTALL.md) y [`AI_SUMMIT/AGENTS.md`](./AI_SUMMIT/AGENTS.md) para instrucciones determinísticas.
